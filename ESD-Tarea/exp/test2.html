Gestión de Objetos

Objetos en JavaScript:
Un objeto es una colección de propiedades, donde cada propiedad tiene un nombre (clave) y un valor. Se crean usando llaves {} y se accede a sus propiedades mediante la notación de punto o corchetes.
Método Object.keys():
Este método devuelve un array con las claves de un objeto. Permite iterar sobre las propiedades del objeto, útil para realizar operaciones como eliminación o modificación.
Método forEach():
Este método es utilizado en arrays para ejecutar una función en cada elemento del array. En nuestro caso, se usa para recorrer las claves obtenidas con Object.keys().
Operador delete:
Se usa para eliminar una propiedad de un objeto. Si la propiedad no existe, no produce error, simplemente no hace nada.
Operador de Propagación (...):
Permite crear una copia superficial de un objeto o array. Útil para evitar mutaciones de objetos originales y trabajar con copias.
Funciones:
Las funciones son bloques de código reutilizables que realizan una tarea específica. Pueden recibir argumentos (como el objeto persona) y devolver un valor (el nuevo objeto con la edad incrementada).
Valores por Defecto (||):
En la línea persona.edad || 0, se utiliza el operador lógico OR para proporcionar un valor por defecto (0) en caso de que edad no exista.
Resumen del Flujo:
1. Se define el objeto.
2. Se añaden hobbies al objeto.
3. Se eliminan las propiedades usando Object.keys() y forEach().
4. Se define y llama a la función para incrementar la edad, que crea un nuevo objeto.
5. Se imprimen los resultados finales.
