Creacion y Manipulacion de Arreglos

Variables que se utilizan en este codigo para esta tarea de estructura de datos es:



bolsaDeLoteria:
Tipo: Arreglo (array). Descripcion: Almacena los numeros de loteria. Comienza vacio y se va llenando con numeros aleatorios, se ordena y se modifica a lo largo del codigo. Metodos Utilizados

Math.random():
 Genera un numero aleatorio entre 0 (inclusive) y 1 (exclusivo). Es util para obtener valores aleatorios.

Math.floor(x):
 Redondea x hacia abajo al numero entero mas cercano. Se usa para convertir numeros decimales en enteros.

push(elemento):
Agrega elemento al final del arreglo. Permite aumentar el tamano del arreglo dinamicamente.

console.log(mensaje):
 Imprime mensaje en la consola. Se utiliza para mostrar informacion al usuario o al desarrollador para depuracion.

sort(comparador):
 Ordena los elementos del arreglo segun el comparador proporcionado. En este caso, se usa para ordenar los numeros de menor a mayor.

splice(inicio, cantidad):
 Modifica el arreglo eliminando o reemplazando elementos. Aqui, se utiliza para eliminar los primeros 3 elementos.

Asi que bueno en si, el flujo del Codigo o snetido del codigo es el siguiente:
 1.-Se crea un arreglo vacio.
2.-Se llenan 10 numeros aleatorios en el arreglo Random.
3.-Se imprime el arreglo original con console.log.
4.-Se ordena el arreglo con el Sort.
5.-Se imprime el arreglo ordenado con en console.log.
6.-Se eliminan los 3 primeros numeros con splice.
7.-Se agregan 5 nuevos numeros aleatorios
. 8.-Se imprime el arreglo final con en console.log.
