Validacion

Variables:
- numerador: El número que se desea dividir.
- denominador: El número por el cual se desea dividir.

Métodos:
- try: Bloque que intenta ejecutar el código dentro. Si hay un error, se pasa al bloque catch.
- throw: Se utiliza para lanzar un error cuando se detecta un problema, como dividir por cero.
- catch: Bloque que maneja el error lanzado, permitiendo controlar la situación sin detener la ejecución del programa.

Flujo del Código:
1. Se llama a la función dividir() con dos números.
2. Se verifica si el denominador es cero.
3. Se intenta realizar la división y manejar errores.
4. Se retorna el resultado o un mensaje de error.
