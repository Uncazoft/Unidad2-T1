Filtrado y Mapeo de Arreglos

Variables:
- productos: Arreglo que contiene productos, cada uno con nombre, precio y categoría.
- categoriaEspecifica: Define la categoría de productos a filtrar.

Métodos:
- filter(): Crea una nueva lista con productos que cumplen ciertas condiciones (por ejemplo, categoría "Ropa" y precio mayor a 100).
- map(): Crea una nueva lista con solo las propiedades deseadas de los productos filtrados (como nombre y precio).

Flujo del Código:
1. Se inicia con un arreglo de productos.
2. Se define la categoría a filtrar.
3. Se aplican filter() y map() para obtener y simplificar la lista deseada.
4. Se imprime el resultado.
